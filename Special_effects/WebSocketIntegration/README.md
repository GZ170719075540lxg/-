# WebSocket 图片接收与显示功能

## 功能说明

此 WebSocket 服务器现在支持：

1. 接收二进制图片数据
2. 自动保存图片文件（带时间戳）
3. 使用 OpenCV 实时显示接收到的图片
4. 返回详细的处理状态给客户端

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行服务器

```bash
python websocket_simple.py
```

## 新增功能详情

### 图片处理流程

1. **接收阶段**: 服务器接收客户端发送的二进制图片数据
2. **保存阶段**: 将图片保存为 `received_[timestamp].png` 格式
3. **显示阶段**: 使用 OpenCV 创建窗口显示图片（2 秒后自动关闭）
4. **反馈阶段**: 向客户端返回详细的处理结果

### 技术实现

- 使用 `cv2.imdecode()` 解码二进制图片数据
- 支持常见的图片格式（PNG, JPG 等）
- 自动时间戳命名避免文件覆盖
- 异常处理确保服务稳定性

## 测试方法

运行测试脚本验证功能：

```bash
python test_opencv_display.py
```

测试脚本会：

1. 首先进行本地 OpenCV 显示测试
2. 然后连接到 WebSocket 服务器发送测试图片

## 使用示例

客户端可以发送任何二进制图片数据，服务器将：

- 在控制台显示接收信息
- 保存图片文件
- 弹出 OpenCV 窗口显示图片
- 返回处理结果给客户端

## 注意事项

- 确保已安装 OpenCV 库 (`pip install opencv-python`)
- 服务器需要 GUI 环境才能显示图片窗口
- 图片窗口会在 2 秒后自动关闭，也可按任意键立即关闭
